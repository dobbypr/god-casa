# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public GitHub issue.

Instead, report it privately by emailing the repository owner directly via the contact information on their GitHub profile, or by using [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) feature.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any suggested fix, if available

We will acknowledge receipt of your report and work to address it promptly.
